## Outdoors SLAM Toolbox Localization and Nav2 Bringup

**See** https://github.com/slgrobotics/articubot_one/wiki/Outdoors-Package-Template

This package launches a full outdoor-capable navigation stack:
- NavSat transform
- EKF fusion
- SLAM Toolbox mapping

Run:
```
  ros2 launch outdoors_loc_nav robot_bringup.launch.py
```
